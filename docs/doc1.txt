This project implements a mini search engine in C++.
The engine supports indexing, ranking, and searching text documents.

An inverted index is built using sharded posting lists.
Each posting stores the document identifier and token positions.

Phrase search requires positional information.
If a document contains the phrase search engine in the correct order, it should rank higher.

BM25 is a probabilistic ranking function.
It improves upon TF-IDF by normalizing document length and term frequency.
